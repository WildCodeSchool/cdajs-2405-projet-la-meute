# Paw Planner

## Stack
- React (ViteJS)
- NodeJS
- Typescript
- Apollo (Client (via Vite) et Server)
- TypeORM
- TypeGraphQL
- PostegreSQL
- Docker
- **Biome** : Biome is an alternative to ESLint and Prettier for ensuring code quality. It provides both linting and formatting in one tool.
